TERMINAL COMMANDS

Available system commands for the hacker terminal

help / commands Display all available commands
list / show all Display all vocabulary words
count / stats Show database statistics
search [term] Search for a specific word
info [word] Detailed information about a word
clear / cls Clear terminal screen
random / zufall Show a random word
theme [dark/light/hacker] Change interface theme
bg [color] Change background color (hex or name)
color [color] Change text color
time Show current time
date Show current date
export Export vocabulary list
matrix Toggle matrix background effect
sound [on/off] Toggle sound effects
shutdown / exit Fake shutdown sequence
